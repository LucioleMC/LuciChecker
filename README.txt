This is a CLI Program you need your cmd.exe for execute this checker

Here's how to use the checker :
1. Open cmd.exe
2. Do the command "cd (the location of the checker)
3. and do lucichecker (syntaxs)

Command Syntaxs :
--file / -f (FILE) : Choose the file that contain words to check (one by lines)
--generate / -g : Generate the file to use for check (not recommanded because very long)
